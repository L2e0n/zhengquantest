# 证券从业资格考试题库系统

一个基于 Vue 3 + TypeScript + IndexedDB 的本地题库系统，支持章节练习、考前点题和真题模拟。

## ✨ 特性

- 📚 **章节练习**：按8个章节系统学习，每节按题型分类
- 🎯 **考前点题**：考前冲刺必做题
- 📝 **真题模拟**：2019-2024年历年真题
- ❌ **错题本**：答错自动收录，支持复习
- 💾 **本地存储**：全部数据保存在浏览器，无需后端
- 📱 **响应式**：支持桌面和移动端

## 🚀 技术栈

- Vue 3 + TypeScript
- Vite
- Element Plus
- Dexie (IndexedDB)
- Pinia

## 📦 安装

```bash
npm install
```

## 🛠️ 开发

```bash
npm run dev
```

## 🏗️ 构建

```bash
npm run build
```

构建产物在 `dist` 目录。
